Big Data
Q Data language 30 mins
Q is a data language in development - for now, it is limited to a data definition language (DDL). Think "JSON/XML schema", but with enhancements. Q comes with a dedicated type system for defining data and a theory, called information contracts, for interoperability with programming and data exchange languages.
Python Tools for Machine Learning 20 mins
Python is perhaps the most utilitarian programming language out there, with extensive coverage in scientific computing: computer vision, artificial intelligence, mathematics, astronomy to name a few. Unsurprisingly, this holds true for machine learning as well.

From the Bayesian Inference to the Bayesian state of mind, this is a great guide for data hackers.

Fast scraping in python with asyncio 20 mins
Web scraping is one of those subjects that often appears in python discussions. There are many ways to do it, and there never seems to be one best way. In this post, George Dubus presents another scraping system based on the new asyncio library : aiohttp. He uses it to write small scrapers that are really fast, and he'll show you how.
Rails - the Missing Parts - Interactors 20 mins
While Rails may be the perfect tool for building an MVP, it isn’t optimised for medium to large codebases. NYC startup Grouper is solving these problems with 3 concepts that they believe should be part of any “Advanced” Rails deployment; Interactors, Policies and Presenters.
An analysis of Facebook photo caching 20 mins
Every day people upload more than 350 million photos to Facebook (as of December 2012) and view many more in their News Feeds and on their friends’ Timelines. Facebook stores these photos on Haystack machines that are optimized to store photos. In this post, Wyatt Lloyd describes the stack and then covers four of Facebook's many findings.

Developing a mobile cross-platform library - Part 2. C++ 35 mins
Kostis Dadamis shares his experiences exploring solutions for developing a mobile cross-platform library, i.e. a single codebase that could be part of mobile apps running under different platforms. This post covers his journey from mobile cross-platform developments tools (PhoneGap, Titanium, and the likes), code porting tools, and WebViews that weren't up to the task, to C++ and JavaScript engines that did work.
